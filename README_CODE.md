# Comment utiliser l'IA contre un joueur en ligne

1 - Aller sur la page internet où la partie se déroule
2 - Lancer Autochess.py
3 - Pour arrêter le code, il faut mettre la souris en haut à gauche ou fermer le kernel (et non l'interrompre seulement)
